
# Oauth Scope

OAuth 2 scopes supported by the API

## Enumeration

`OauthScope`

## Fields

| Name | Description |
|  --- | --- |
| `HTTPS_WWW_GOOGLEAPIS_COM_AUTH_CALENDAR_READONLY` | See and download any calendar you can access using your Google Calendar. |
| `HTTPS_WWW_GOOGLEAPIS_COM_AUTH_CALENDAR_EVENTS` | View and edit events on all your calendars. |
| `HTTPS_WWW_GOOGLEAPIS_COM_AUTH_CALENDAR` | See, edit, share, and permanently delete all the calendars you can access using Google Calendar. |
